# PowerShellモジュールをシステム全体にインストールする方法

PowerShellモジュールをシステム全体で利用可能にするための手順について説明します。

## 概要

PowerShellモジュールは通常、ユーザー固有の場所にインストールされますが、システム全体で利用したい場合は管理者権限でインストールする必要があります。

## 手順

### 1. 管理者権限でPowerShellを起動

```powershell
# 管理者としてPowerShellを開く
Start-Process PowerShell -Verb RunAs
```

### 2. インストールスコープを指定してモジュールをインストール

```powershell
# システム全体にインストール
Install-Module -Name ModuleName -Scope AllUsers
```

## まとめ

この方法により、すべてのユーザーがモジュールを利用できるようになります。
